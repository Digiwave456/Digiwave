Инструкция по запуску:
(Требования: PHP 8.1+, Composer, Node.js 18+, MySQL/PostgreSQL,Laravel 10.48.23+)
Установка зависимостей:
composer install --optimize-autoloader --no-dev

npm install

параметры в .env:
APP_NAME=Laravel
APP_ENV=local
APP_KEY=
APP_DEBUG=true
APP_URL=http://localhost

LOG_CHANNEL=stack
LOG_DEPRECATIONS_CHANNEL=null
LOG_LEVEL=debug

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel
DB_USERNAME=root
DB_PASSWORD=
